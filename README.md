## Machine Learning Repository
This repository contains multiple notebook projects exploring machine learning, regression, classification, clustering algorithms, ETL pipelines, and data integration.
Each notebook demonstrates step-by-step implementation, model evaluation, and data analysis using Python.
All work is developed in .ipynb notebooks, allowing experimentation, visualization, and reproducible workflows.

## How to Run the Notebooks
```bash
git clone https://github.com/camzil/ML
cd ML
jupyter notebook
```

Open any notebook to view or execute the workflow.
